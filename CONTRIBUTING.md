# Contributing

Thanks for considering a contribution.

Useful contributions include:

- improving browser compatibility;
- adding privacy-safe synthetic examples;
- improving keyboard and screen-reader behavior;
- making redaction export more reliable;
- documenting limitations clearly.

Before opening a pull request:

- run `npm run lint`;
- run `npm run test`;
- run `npm run build`;
- run `python scripts/validate_public_assets.py`;
- do not commit real photos, personal documents, screenshots, API keys, or private paths.
