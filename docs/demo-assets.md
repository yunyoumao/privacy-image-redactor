# Demo Assets

This repository uses only synthetic demo assets.

## Asset Types

- `examples/before/*.svg` and `examples/after/*.svg` are deterministic fixtures used by README examples and verification checks.
- `examples/showcase/privacy-image-redactor-showcase.png` is an AI-generated product mockup used as the README showcase image.

## Generation Boundary

The showcase image was generated from a text prompt describing a local-first privacy image redaction editor. It uses fictional interface content and fake example fields. It is not a screenshot of a real meeting, lab record, company page, personal document, thesis figure, or unpublished project.

The image is for public presentation only. The SVG fixtures remain the source-controlled samples for reproducible testing and app behavior.

## Review Rule

Before adding or replacing public demo images:

1. Use synthetic or fictional content only.
2. Avoid real faces, real documents, real company screens, real research data, and real contact details.
3. Inspect the image manually before committing it.
4. Keep generated showcase images separate from reproducible fixtures.
